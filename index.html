.size{} Shortcuts

You often need to set width and height for an element.
Use the .size mixin as follow: 

p {
    .size( 70px );
}

@w:70px 

p {
    .size( 150px, 80px );
}

@w:150px, @h:80px

.block{} Shortcuts

You often need to set an item to act as a block element with the block rule.
Use the .block mixin as follow: 

p {
    .block;
}

@w:auto 

p {
    .block( 70px );
}

@w:70px 

p {
    .block( 150px, 80px );
}

@w:150px, @h:80px

.block with .size!

.block mixin sets up the block CSS property but is also able to set box size using the .size mixin!

.inline-block{} down to IE6!

LessMixin supplies an .inline-block mixin, use it when needs to define an inline-block item guaranteeing retro-compatibility to IE8, IE7 and IE6!

inline-block

inline-block( 150px )

inline-block( 150px, 90px )

You can use the .iblock shortcut instead of .inline-block mixin!

.mask{} mixin

You often need to set an item to act as a mask element with the overflow rule.
Think about sprites, layout cards, etc...

Use the .mixin mixin as follow: 

p {
    .mask;
}

p {
    .mask( 80px );
}

p {
    .mask( *, 50px );
}

p {
    .mask( 80px, 80px );
}

.inline-mask{}

You can play with inline-masks using the .inline-mask mixin or it's shortcut .imask.

This mixin works just like the .mask with the same options to set mask's width and height.

Positioning mixin

LessMixin supplies a set of utilities mixins to sets up relative and absolute positioning of DOM elements.

.rel force an element to act as a relative positioning container so all children can be absolute positioned from the top-left corner of container itself.

.abs and it's variations force an element to position absolute to the first relative parent element.

.abs( @vSpace, @hSpace) takes two optional params.

@vSpace
sets up the distance from the top/bottom corner
@hSpace
sets up the dinstance from the left/right corner

There are some variations of the .abs mixin to change the corner from which position is calculated.

.abs()
top/left corner
.abs-r()
top/right corner
.abs-b()
bottom/left corner
.abs-br()
bottom/right corner

.abs( null, 90px );

.abs( 40px, 90px );

.sprite{} Techniques

Sprites allow you to optimize background images weight packing them into a single image.
Creating CSS code for sprites may be complicated but with LessMixin you can easily set up some CSS sprited buttons, even with two or tree states!

Less code to implement above buttons may be something like following: 

btn {
    .sprite( "img/sprite.png", @width, @height, @x-pos, @y-pos )
}

btn-hover {
    .sprite( "img/sprite.png", @width, @height, @x-pos, @y-pos, @y-pos-hover )
}

btn-hover-active {
    .sprite( "img/sprite.png", @width, @height, @x-pos, @y-pos, @y-pos-hover, @y-pos-active )
}
